# 🎨 Simple CSS Layout Demo

This repository contains a basic HTML and CSS project that demonstrates how to create and style elements using classes and containers. 
It is designed as a beginner-friendly example to understand core frontend concepts.

## Overview
The project showcases a simple layout with multiple styled boxes inside a container and one box outside it. 
It helps in understanding how CSS classes are applied and how layout structure works.

## Features
- Clean and minimal HTML structure  
- Reusable CSS classes for styling  
- Container-based layout design  
- Beginner-friendly code
